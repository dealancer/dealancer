kube-httpcache

kube-httpcache is an Open Soruce Varnish Reverse Proxy on Kubernetes written in Go. I took an active part in adding cluster feature and wrote the Signaller component for broadcasting cache invalidation messages.

Browse code repo
Kubernetes Navigator training

Kubernetes Navigator is an online introductory 4-hour Kubernetes training that will help you understand basic features and concepts of Kubernetes such as declarative YAML manifests, availability, autoscaling, load balancing, and automated deployment. You will get hands on experience working with Kubernetes and creating basic resources.

Check out training files
Druml - Drupal multisite management tool

Druml is a Drupal multisite tool that helps to maintain hundreds of Drupal websites in the mulstisite Drupal installation. It has been developed by me at Georgetown University. Druml uses Bash and Drush. It also works nicely with Acquia Cloud Platform.
